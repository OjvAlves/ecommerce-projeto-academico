# 🛒 E-commerce - Projeto Acadêmico

## 📝 Sobre o Projeto
Este é um projeto de uma aplicação de e-commerce totalmente funcional, desenvolvido como parte do critério avaliativo do **2º semestre do curso de Engenharia de Computação na UFMG**. 

O objetivo principal foi aplicar conceitos de desenvolvimento de software, estruturação de dados e experiência do usuário (UX) em um cenário real de mercado.

---

## 🚀 Funcionalidades Principais
* **Catálogo de Produtos:** Visualização de itens disponíveis com filtros.
* **Carrinho de Compras:** Adição, remoção e atualização da quantidade de produtos.
* **Simulação de Checkout:** Fluxo de finalização de compra (simulado).
* **Responsividade:** Interface adaptada para dispositivos móveis e desktops.

---

## 🛠️ Tecnologias Utilizadas

* **Back-end:** C++
* **Ferramentas:** Git e GitHub
* **POO:** paradigma de programação orientada a objetos para estruturação e arquitetura de software
